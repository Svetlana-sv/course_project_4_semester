Курсовой проект «Веб-сайт для доставки продуктов»

В рамках данного курсового проекта была произведена доработка web-приложения для доставки продуктов.
Целью данной курсовой работы является модернизация многостраничного сайта, который представляет из себя структурированную систему, предоставляющая покупателям полную информацию о продуктах, с возможностью оформления заказа через личный кабинет.

Курсовой проект «Веб-сайт для доставки продуктов»

В соответствии с поставленной целью в курсовой работе поставлены следующие задачи:

1. Произвести анализ отечественных и зарубежных аналогов.
2. Разработать макет для web-приложения.
3. Разработать web-приложение с использованием системы контроля версий git.
4. Описать особенности функционирования и провести тестирование разработанного сайта.

Для разработки веб-сайта были использованы следующие программные средства:

Среда разработки Visual Studio Code.
Фреймворк Vue.js.
Язык разметки гипертекста html и css.
JavaScript.
PHP.
PhpMyAdmin.
Figma.
Postman.
Дистрибутив XAMPP.
